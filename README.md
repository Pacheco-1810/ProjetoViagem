# ViajeMais

Projeto acadêmico de um site de turismo desenvolvido com HTML, CSS, JavaScript e Bootstrap.

## Funcionalidades

- Cadastro de usuários
- Login e logout
- Carrinho de compras
- Adição e remoção de pacotes
- Controle de acesso ao carrinho
- Exibição do usuário logado

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Armazenamento

O sistema utiliza:

- `localStorage` para armazenar os dados do cadastro
- `sessionStorage` para controlar a sessão do usuário logado

## Objetivo

Aplicar conceitos de desenvolvimento web, manipulação do DOM, armazenamento local e versionamento com Git/GitHub.
