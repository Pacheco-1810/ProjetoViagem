# 🌎 ViajeMais Turismo

O **ViajeMais Turismo** é um site desenvolvido para uma agência de viagens fictícia, com o objetivo de apresentar destinos turísticos, divulgar pacotes de viagem e proporcionar uma experiência simples e intuitiva para os usuários interessados em planejar sua próxima aventura.

## ✈️ Funcionalidades

### Página Inicial
- Logo da agência em destaque.
- Menu de navegação responsivo.
- Seção promocional para atrair clientes.
- Carrossel de imagens com destinos turísticos.
- Rodapé com informações da empresa.

### Página de Pacotes
- Exibição de pacotes turísticos em formato de cards.
- Imagens ilustrativas dos destinos.
- Informações básicas sobre cada viagem.
- Seleção da quantidade de viajantes.
- Carrinho de compras dinâmico.
- Cálculo automático do valor total.
- Remoção de itens adicionados ao carrinho.

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Google Fonts
